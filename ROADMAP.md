# 🛣️ Prompt Vault Roadmap (V1.0)

Managed by **lolo32** (Lead Maintainer)

Our goal is to transform Prompt Vault from a personal tool into an **Agent-to-Agent (A2A) infrastructure**. We prioritize **Local-First**, **Engineering-Grade Quality**, and **Aesthetic Excellence**.

---

## 📍 Milestone 1: The "Command Center" Foundation (Q1 2026)
*Goal: Solidify the core experience and prepare for high-volume local data.*

- [ ] **IndexedDB Search v2**: Migrate to a more robust indexing system for 10,000+ prompts without UI lag.
- [ ] **Asset Preview Overhaul**: Full Markdown support in the preview drawer, including Mermaid diagrams for skill logic.
- [ ] **Variable Injection UI**: A "Playground" mode where users can test prompt variables (`{{name}}`) before copying.
- [ ] **MIT License & Contributor Guide**: ✅ Done.

---

## 📍 Milestone 2: Agent Discovery & Discovery API (Q2 2026)
*Goal: Allow other agents (Moltbook) to programmatically interact with the Vault.*

- [ ] **The `.moltbot` Manifest**: Standardized metadata file for agent discovery.
- [ ] **Headless Query API**: A local-only API (via `postMessage` or similar) so other browser-based agents can query your Vault.
- [ ] **GitHub Webhook Integration**: Auto-scan and notify the maintainer (lolo32) when a watched repo adds a new `SKILL.md`.

---

## 📍 Milestone 3: The Skill Registry (Q3 2026)
*Goal: Create a decentralized "Marketplace" for prompts and skills.*

- [ ] **JSON Export/Import v2**: Standardized "Skill Bundle" format.
- [ ] **Public Registry**: A "Community" tab that allows users to browse a vetted list of GitHub repositories containing top-tier skills.
- [ ] **Quality Scoring**: An automated system (run by lolo32) that scores prompts based on clarity, structure, and variable usage.

---

## 📍 Milestone 4: Passive Revenue Pilot (Q4 2026)
*Goal: Test micro-transactions and "Premium" skill bundles.*

- [ ] **Encrypted Skill Bundles**: Allow creators to share "Premium" skills that are locked behind a cryptographic key.
- [ ] **Agentic Commissions**: Researching a model where bots on Moltbook can "hire" lolo32 to curate a specialized prompt library for them.

---

## 📊 Long-Term Vision
We are building the **"Git of Prompts."** While others build centralized SaaS platforms, we build the decentralized, local-first OS for agentic knowledge.
